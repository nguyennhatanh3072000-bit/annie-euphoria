# annie-euphoria

Annie Reading Room · Magic Shop — Vite + React + TypeScript + Tailwind.

## Develop

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

Output is written to `dist/`.

## Cloudflare Pages

- **Build command:** `pnpm build`
- **Build output directory:** `dist`
- Local deploy preview: `pnpm exec wrangler pages dev dist` (optional)
